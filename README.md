# Geometry 7 Class Telegram Tutor

Telegram-бот по геометрии 7 класса.

Что умеет:
- объясняет темы простыми словами
- не даёт готовые решения
- помогает найти идею доказательства
- принимает PDF, DOCX, JPG, PNG
- даёт тренировку на 5 заданий
- даёт викторину с уровнями сложности: лёгкий / средний / сложный
- отдельный режим: 5 правильных подряд
- звёздочки, уровни, достижения, задание дня

## Запуск

1. Создайте бота у @BotFather
2. Добавьте токен в Railway Variables:
   - `BOT_TOKEN=ваш_токен`
3. Установите зависимости:
   - `pip install -r requirements.txt`
4. Запустите:
   - `python bot.py`

## Railway

В Railway нужен только один обязательный secret:
- `BOT_TOKEN`

Для OCR на картинках добавьте apt packages:
- `tesseract-ocr tesseract-ocr-rus`
